# Hi, I'm Rahmad

Civil engineer by day, building software for the construction industry by night.

I work in civil engineering and got into coding because I kept running into problems that spreadsheets couldn't solve. Now I build tools that help construction teams manage projects, estimate costs, and make better decisions — mostly with TypeScript and Python.
